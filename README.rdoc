== README

* install ruby 2.2.3
* install postgres
* gem install bundler

* cd beer-graph
* bundle

* edit config/database.yml for your postgres installation

* rake db:create
* rake db:migrate
* rake db:seed

* rails s

* open a browser and navigate to http://localhost:3000

* You will see GraphiQL - a react component for GraphQL

* try the following query 

{ 
  beers{
    name
    description
    style {
      name
    }
  }
}
